/bin/    : basic user programs
/boot/   : boot files and the linux kernal 
/dev/    : Device related files
/etc/    : System and Configuration files
/home/   : user profiles and personal files
/lib/    : software libraries 
/media/  : mount folder for removal media
/mnt/    : temporary mount point 
/opt/    : Third party software
/proc/   : Linux systtem details
/root/   : root user's home directory 
/run/    : volatile runtime data
/sbin/   : system binaries
/srv/    : files for server applications
/tmp/    : temporary directory files
/usr/    : user shared files and folder 
/usr/bin/: kali tools (binaries)
/var/    : logs that are generated by daemons
/var/www/html: apache web server
